# Delivery_Frontend

## 🔗 Repositórios do Projeto

Este sistema de delivery é dividido em dois repositórios:

- 🖥 Backend: https://github.com/Rochagx/Delivery_backend
- 🎨 Frontend: https://github.com/Rochagx/Delivery_Frontend

Projeto de **frontend para sistema de delivery** desenvolvido com **Angular, TypeScript, HTML e CSS**.

## Funcionalidades

- Listagem de produtos
- Adição de itens ao carrinho
- Criação de pedidos
- Atualização de status dos pedidos
- Gerenciamento de carrinho e histórico de pedidos

## Tecnologias utilizadas

- Angular
- TypeScript
- HTML
- CSS
- API
